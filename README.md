# Swept AABB Collision Example
Some C++ files extracted from my voxel engine project, for public use.

Known Issues:
At very low framerates, such as below 5 FPS or so, the player can phase a few blocks into the floor, but no further, I suspect this is due to how I'm handling physics updates in my voxel engine, I am currently unsure if the source files in this repository are related or problematic to this issue.
